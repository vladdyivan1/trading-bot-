# trading-bot-

This repository contains the **IBKR LLM Trading Assistant** MVP.

See [ibkr-llm-trading-assistant/README.md](ibkr-llm-trading-assistant/README.md) for installation, configuration, and usage.

## TradingView script (SPY)

A TradingView Pine strategy with built-in P/L reporting is available at:

- `tradingview/spy_profitable_pl_strategy.pine`

Open TradingView Pine Editor, paste the script, and run it on the SPY chart to see performance in Strategy Tester.
